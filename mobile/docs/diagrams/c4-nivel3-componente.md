## Componentes por Camada

### Controllers (REST API)
| Componente | Endpoints | Descrição |
|------------|-----------|-----------|
| `CursoController` | `/api/cursos/**` | CRUD de cursos e progresso |
| `RankingController` | `/api/ranking/**` | Ranking geral e por departamento |
| `ConquistaController` | `/api/conquistas/**` | Badges e desbloqueios |
| `AuthController` | `/api/auth/**` | Login, JWT, refresh |

### Services (Lógica de Negócio)
| Componente | Responsabilidade |
|------------|------------------|
| `CursoService` | Listagem, detalhes, progresso de cursos |
| `GamificacaoService` | XP (+10/tarefa), badges, ranking, níveis |
| `ProgressoService` | Registro de conclusão, cálculo de % |
| `OnboardingService` | Trilhas guiadas, checklists |

### Repositories (Acesso a Dados)
| Repositório | Queries Principais |
|-------------|-------------------|
| `UsuarioRepository` | `findByEmail`, `findAllByOrderByXpTotalDesc` |
| `CursoRepository` | `findByTipo`, `findByIdWithModulos` |
| `ProgressoRepository` | `countConcluidasByUsuarioIdAndCursoId` |
| `ConquistaUsuarioRepository` | `findByUsuarioId` |
